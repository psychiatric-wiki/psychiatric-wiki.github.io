# 本站的大致愿望以及关于本站的一些基本信息

## 本站的大致愿望

本站的愿望是针对当前精神科医疗体系的不足，即体系的对立、非人化、漠视和姑息，以及其无法满足人们对精神亚健康阻滞、有效治疗和急性干预的迫切需求。鉴于现行体系难以改变，本站致力于组织社群性的精神科医疗自救。核心目标是建立非对立、平等、人道的医患关系和互助支持体系，确保急需支持者能获得支持。实现路径包括建立实际的社群支持体系、普及和共享精神科医疗资源，最终通过平等、相互理解和信任，实现真正有效的治疗与康复。
