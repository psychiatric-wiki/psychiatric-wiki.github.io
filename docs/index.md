  - title: 现在的精神科医疗体系做了什么？
    icon: 🤨
    details: 现在的精神医疗体系以对立且高于而漠视的非人化的姿态对待患者，以姑息的态度对待自己的职业与患者的治疗与康复。对处于精神亚健康状态的人的精神状态恶化几乎起不到任何阻滞效果；对已患有精神疾病而急需精神科医疗支持的人要么保守至极甚至毫无作为，要么借治疗的名义反而却损害患者的精神；甚至有时伙同加害者，作为精神疾病与创伤的创造者。
  - title: 我们要做什么？
    icon: 🤲
    details: 我们要恰恰反其道而行之！只有非对立性的、平等的、人道的医患关系才能实现真正有效的治疗，才能杜绝姑息与加害。在使现行精神科医疗体系达成这样改变存在极度的困难的现实状况下，我们只能社群性地自己来接过这份责任：在平等且互助的社群中组织出适当的精神科医疗支持体系，使能够有能力提供精神科医疗支持的人能提供支持；使急需精神科医疗支持的人能获得支持。这必然通过体系中双方平等的相互理解与信任来实现。同时，这也必然则要求体系中双方差异性的减小，即精神科医疗资源的普及。
  - title: 我们怎么才能做到？
    icon: 🧐
    details: 正如上所述，我们必须组织社群的精神科医疗自救。这除了要求平等与理解与信任以外，更要求事实上的社群医疗支持体系的建立以及精神科医疗资源的普及与共享。这正是本站所致力于实现的目标。
